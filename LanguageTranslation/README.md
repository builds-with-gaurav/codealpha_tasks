# 🌐 Language Translation Web App

A responsive Language Translation Web Application built using **HTML**, **CSS**, and **JavaScript**. The application allows users to translate text between multiple languages through an online translation API, providing a clean and user-friendly interface.

## 🚀 Live Demo

🔗 https://builds-with-gaurav.github.io/codealpha_tasks/LanguageTranslation/

---

## 📌 Features

- 🌍 Translate text between multiple languages
- 🔄 Swap source and target languages
- 📋 Copy translated text
- 🎤 Text-to-Speech support (if available)
- 📱 Responsive design for all devices
- ⚡ Fast and easy-to-use interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Translation API
